# UCounter

UCounter, ekran üstünde CPU ve RAM kullanımını gösteren, konum ve görünüm seçenekleri sunan bir Windows overlay aracıdır.

## Ne Yapar

- CPU ve RAM kullanımını izler
- değerleri yazı ya da grafik biçiminde gösterebilir
- ekranın farklı köşelerine yerleşebilir
- sistem tepsisine küçülerek çalışabilir

## Teknolojiler

- C#
- Windows Forms
- Managed DirectX
- WMI
